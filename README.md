# PDF to Markdown 변환 도구 비교 평가

학술 논문 PDF를 마크다운으로 변환하는 4가지 도구의 품질을 비교 평가한 프로젝트입니다.

## 평가 대상

- **원본 PDF**: TV-RAG (ACM MM'25 학술 논문)
- **변환 도구**: markitdown, pdftotext, pymupdf, mineru

## 평가 결과 요약

| 순위 | 도구 | 총점 | 등급 | 핵심 특징 |
|:----:|------|:----:|:----:|-----------|
| 1 | **mineru** | **92/100** | **A** | 수식 LaTeX 완벽, 표 HTML 보존 |
| 2 | pymupdf | 54/100 | D | 이미지 참조 우수, 수식/표 손실 |
| 3 | pdftotext | 39/100 | F | 텍스트만 추출, 구조 손실 |
| 4 | markitdown | 14/100 | F | 심각한 노이즈, 사용 불가 |

## 평가 루브릭

7개 기준으로 총 100점 만점 평가:

| 기준 | 배점 | 설명 |
|------|:----:|------|
| 텍스트 정확성 | 20 | 원문 추출 정확도 |
| 구조 보존 | 20 | 제목/섹션 마크다운 문법 |
| 수식 처리 | 15 | LaTeX 형식 변환 |
| 표 처리 | 15 | 테이블 구조 보존 |
| 이미지 참조 | 10 | 마크다운 이미지 참조 |
| 가독성 | 10 | 렌더링 편의성 |
| 노이즈/아티팩트 | 10 | 불필요한 문자 여부 |

## 결론

**mineru**가 92점으로 압도적 1위. 수식(LaTeX)과 표(HTML) 처리에서 유일하게 만점을 받아 학술 논문 변환에 최적.
